The Bose Wave codes are fairly non-standard.  These codes match the 18 button remote. At this time I do not know which version of Wave I have.

The TCL Roku TV model:50S425 remote repeats some keys with one initial code and a different code for repeat. 
The repeat code is resent until the button is no longer pressed. The repeats may be useful for a manually operated remote, 
but are useless for automation, so I did not include them. Some keys, like power, do not send repeats. 

The YAML file is ESPHome code for an IR blaster ESP8265. Extract for your purposes.
